# Mini-Project 2 Proposal
## What I'm Building
- I'm building a web tool that is an interactive decision making tool that helps the user pick a place to eat by randomly selecting a restaurant from a user-generated list.

## Who it's for/Why
- This is for anyone who can't decide where to eat or what meal to have. It allows for them to enter some foods they may be craving and then randomly selects one for them.

## Core Features
1. Dynamic List Building: users add restaraunt name into an input field and clikc add to build a custom wheel.
2. The Spin Interaction: A spin button that triggers a visual shuffling through the list before landing on a winner
3. Visual Feedback: The winning choice will be highlighted with a unique style (bold color, gold border, winner badge) to make it clear which is the selected option.
4. Reset Function: A clear button that allows user to wipe the array and reset the UI so a new list can be built.
